# 使い方
## ①pythonをインストールする。→https://www.python.org/
(すでにダウンロードしてある場合は飛ばしてOK)

## ②「初めだけ実行してください.bat」を実行する

## ③親パスの設定
・parent.batの親パスを設定する

## ④使用する音声ファイルの設定
・script/setPath.batの、「音声ファイル名」に自動化したい音声ファイルを設定する

## ⑤main.batのショートカット作成
maih.bat - ショートカットを右クリック→「プロパティ」→「ショートカット」→「ショートカットキー」を設定する

音声ファイルの数だけscriptフォルダを増やせば、多くのショートカットを作成することができます。

# 注意
・ショートカットキーを押す時は、マウスカーソルをレイヤー上に置いてお願いします。

・実行中はマウスカーソルを動かさないようにお願いします
(マウスカーソルをプログラムで動かして音声ファイルを追加するため)
